## `!filme`

Mostra informações do filme ou série que está sendo assistido na live.

As informações mostradas são:
- Nome
- Ano de Lançamento
- Duração
- Sinopse
- Link do [IMDb](https://www.imdb.com/)

---

## `!filme assistir`

Marca um filme ou série como "assistindo" e mostra informações como:

- Nome
- Ano de Lançamento
- Duração
- Horário de início
- Link do [IMDb](https://www.imdb.com/)
- Sinopse

<ModWarn></ModWarn>

<Note>Caso a sinopse tenha mais que 350 caracteres, a Sayurita irá resumir a sinopse original.</Note>

### Sintaxe

```bash
!filme assistir <nome_do_filme>
```

### Exemplo

```bash
!filme assistir wall-e
```

### Props

<ResponseField name="nome_do_filme" type="string" required>
  O nome do filme ou série que deseja consultar.
</ResponseField>

---

## `!filme info`

Mostra informações de qualquer filme ou série.

As informações mostradas são:
- Nome
- Ano de Lançamento
- Duração
- Avaliação *(no TMDB)*
- Link do [IMDb](https://www.imdb.com/)
- Sinopse

<ModWarn></ModWarn>

### Sintaxe

```bash
!filme info <nome_do_filme>
```

### Exemplo

```bash
!filme info wall-e
```

### Props

<ResponseField name="nome_do_filme" type="string" required>
  O nome do filme ou série que deseja consultar.
</ResponseField>
